# Regular expressions
```
g   - global - find all occurrences
i   - insensitive
m   - multiline
()  - groups
|   - or
\   - insert special character
^   - start with something
$   - finish with something
[^] - negation
```
<b>Quantifiers:</b>
```
+   - 1 or N occurrences {1,}
*   - 0 or N occurrences {0,}
?   - 0 or 1 occurrences {0,1}
```
